# BalconyPlanter
Some .stl files for a planter I 3D printed to accommodate my balcony. Please use if helpful!

- Just use your slicer to convert from .stl to .gcode or whichever file type your printer accepts. I use an Ender-3 V2 and this worked for me. I did a 20% infill density for the hooks just to make sure they could hold the weight of the soil.


03May2026
I have uploaded this V3 version to accommodate my 3D Printer better. I gave up 20mm of planter length on the top layer so I could print this flat on the bed. This resulted in a quicker printing time as well as less material used due to significantly less supports.

Changes:
* Planter depth increased
* Planter length slightly reduced (by 20mm)
